# Fact-Check

一个面向 iOS 16+（目标 iOS 26） 的 SwiftUI 事实核查应用示例，专注于将用户陈述与正文、网页链接一并联网核查，并给出可信度与行动建议。

## 功能
- **快速输入**：提供陈述、正文/聊天记录文本框、上下文与来源链接输入框，方便用户描述待核查的信息。
- **联网交叉核查**：模拟至少 20+ 媒体、博主与官方渠道的信源交叉对比，对复杂信息按最高可能性回复真/假倾向。
- **结果卡片**：以颜色和图标展示判定（已证实 / 存疑 / 无法核实），列出证据摘要、来源链接、综合可信度进度条与归档时间。
- **历史记录**：每次核查都会加入顶部，便于查看最近的验证结果。

## 运行
1. 使用 Xcode 15+ 打开仓库，`Package.swift` 已设置平台为 iOS 16 及以上（面向 iOS 26）。
2. 选择 `FactCheckApp` 可执行目标并在模拟器或真机上运行。
3. 在主界面输入陈述与正文内容（可粘贴网页链接），点击“立即核查”查看联网交叉结果卡片。

> 本仓库在 Linux 环境下无法直接运行 SwiftUI 预览或构建 iOS 应用，请在 macOS + Xcode 环境中打开。
